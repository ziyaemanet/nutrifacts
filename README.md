# Nutrifacts

A small app to display info from:
https://ndb.nal.usda.gov/ndb/doc/index

##### Usage

1. npm i
2. npm run build
3. npm start <--OR--> npm run dev

##### Live

A live version can be found here:

https://ziya-emanet-nutrifacts.herokuapp.com/
